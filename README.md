# KitchenRoad-Frontend

Repositório Frontend

### Baixando e rodando o projeto:

#### 1. Clone o repositório:

```
$ git clone https://github.com/Kitchen-Road/KitchenRoad-Backend.git
```

#### 2. Acesse o diretório do app:

```
$ cd app
```

#### 3. Instale o node:

```
$ npm install
```

#### 3. Inicie o app:

```
$ npm start
```
